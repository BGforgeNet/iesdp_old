WAV file format

Applies to:
BG1, BG1: TotS, BG2, BG2: ToB, PST, IWD, IWD:HoW, IWD:TotL, IWD2


General Description
WAV Files are used for player character sounds ("soundsets") and for effects sounds in game. They are in located the sounds folder under the game directory. The naming conventions for soundsets are documented in the game readme files. WAV files should be 22050 Hz, 16 bit and mono. WAV files can be created using various programs, and converted to ACM or WAVC. A longer description of the standard WAV is pending.
