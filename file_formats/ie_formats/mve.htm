MVE file format

Applies to:
BG1, BG1: TotS, BG2, BG2: ToB, PST, IWD, IWD:HoW, IWD:TotL, IWD2

General Description
This file format describes the movies played during the game.
The format is strictly copyrighted by Interplay, and so the format will not be detailed here. Movies can only be played by the engine when they are stored in a BIFF file.
