scenexx.2da

Defines the textbox shown at occasional points during the storyline.
First name (in our example Ch01aJon) determines BMP reference for text screen.
The first column is filled only by identifiers.
The second column lists the title of the textbox screen.
The third column lists the main text of the textbox screen.

In BG2EE we have (scene01.2da):
2DA V1.0

DEFAULT                 -1              -1
GOOD_REPUTATION         -1              17698

BAD_POWER               NONE            NONE
