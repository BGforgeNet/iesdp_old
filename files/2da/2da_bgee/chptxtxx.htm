chapterxxx.2da

This file defines how to show chapter introduction text.
First name (in our example ch05aund ) determines MOS reference for chapter screen.
The first column is filled by race names. Not all tables include rows for races or classes.
Later there are two columns that could be filled by strrefs from Dialog.tlk.
